Bike Sharing With Spark

Jan 3, 2017

I have been spending a lot of time learning about Spark recently, so I decided to do a simple regression task using Scala and Spark on the Bike Sharing Dataset. It is an interesting (albeit relatively small) dataset and provided a good way to play around with and become more comfortable using the APIs that Spark provides for working with data and building predictive models.

You can check it out here Bike Sharing With Spark

Overall I think I learned a great deal, and was able to learn a lot about how Spark works, and doing these kind of analyses is always a good time. I might revisit this dataset again in the future, but in the context of an anomaly detection task; the thought being that one can use the number of bicycles used for the day to detect events in the city (Washington DC in this case).
